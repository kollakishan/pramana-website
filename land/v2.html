• CODEFREAK


This is a programming event conducted by CSE and IT students. The event comprises of three rounds:
.@ First round:this will be a paper exam which contains basic aptitude and programming questions.
.@ Second round: this will be an online exam which will test the programming skills of the contestant.
.@ Third round: this will be a coding exam where contestants should write programs which will satisfy the constraints. Exciting prizes will be given to the winners and certificates for the participants.
Reg. Fee 130(per head), 200 (team of two).
